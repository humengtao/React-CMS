# React-CMS
Content Manage System by React-Redux-Webpack-NodeJs-MySql 

## Install
###1.Clone the source code.
```bash
git clone https://github.com/humengtao/React-CMS.git
```
###2.Install Dependencies
```bash
npm install
```
###3.Build Assets
```bash
npm run build:dev
```
###4.Run ApiServer
```bash
npm run api
```
###5.Run WebServer
```bash
npm run server
```
Enjoy !!
